**CODM ProStats - Call of Duty: Mobile Dashboard**

CODM ProStats is a sleek, responsive web dashboard designed for Call of Duty: Mobile players to showcase their in-game statistics, rank progress, and weapon camo achievements. Built with a futuristic "gaming" aesthetic, it features high-performance visuals including particle effects and video backgrounds.

**🚀 Features**

* **Player Profile Display**: Showcases core player data such as In-Game Name (IGN), UID, Clan affiliation, and total Likes.
* **Mode-Specific Stats**: Detailed breakdown of performance across Multiplayer (MP), Battle Royale (BR), and DMZ modes, including rank, win streaks, and K/D ratios.
* **Weapon Category Navigation**: A dedicated grid to explore different weapon classes (AR, Sniper, LMG, SMG, etc.) and track specific progress.
* **Animated Camo Progress**: Visual progress bars with custom shimmer effects for Gold, Platinum, Diamond, and Damascus camos.
* **Immersive Experience**: Includes a background video trailer on the hero section and a toggleable ambient soundtrack.
* **Responsive Design**: Fully optimized for mobile devices with a custom hamburger menu and flexible grid layouts.

**🛠️ Tech Stack**

* **HTML5**: Semantic structure for the dashboard.
* **CSS3**: Custom styling using CSS Variables, Flexbox, Grid, and keyframe animations for the "glow" and "shimmer" effects.
* **JavaScript (Vanilla)**: Handles mobile menu toggling, active scroll highlighting for navigation, and background music controls.
* **Google Fonts**: Utilizes `Orbitron` and `Rajdhani` for a modern, sci-fi gaming UI.
* **Font Awesome**: Iconography for the logo and UI elements.

```text
├── ...              # Other category-specific HTML files

```

## 🎨 Design Highlights

* **Glassmorphism**: The dashboard uses `backdrop-filter: blur(8px)` and semi-transparent backgrounds to create a layered, modern feel.
* **Dynamic Animations**: Particles float in the background while the crosshair logo pulses, enhancing the interactive feel.
* **Visual Hierarchy**: High-contrast colors like `#ff4444` (Primary Red) and `#00d4ff` (Secondary Cyan) are used to draw attention to critical stats.
